# Number Guessing Game

A Python-based command-line game where players try to guess a randomly generated number within a selected range. The game provides hints indicating whether the guess is too high or too low, tracks the number of attempts, stores the best score during the session, and allows users to replay. This project demonstrates the use of loops, conditionals, functions, input validation, and Python's random module.

## Features

* Multiple difficulty levels
* Random number generation
* Higher/Lower hints
* Attempt counter
* Best score tracking
* Replay option
* Input validation
* Interactive gameplay

## Technologies Used

* Python 3
* Random Module
